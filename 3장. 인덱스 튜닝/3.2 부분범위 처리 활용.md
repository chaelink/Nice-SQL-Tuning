### 3.2 부분범위 처리 활용

#### 3.2.1 부분범위 처리

부분범위 처리
- 사용자로부터 fetch call이 있을 때 마다 일정량 씩 나누어 전송하는 것
- 데이터를 전송하는 사이즈인 arraysize는 클라이언트 프로그램에서 설정
- ex) arraysize가 10일 때 레코드 100개 응답 시 fetch call은 10번 만 발생
- 정렬 조건이 있으면? 전체범위 처리
  - 인덱스 등을 활용해 이미 정렬되어 있는 경우 제외
 - n tier 환경에서의 부분범위 처리는? - top N 방식으로 가능


#### 3.2.3 OLTP 환경에서 부분범위 처리에 의한 성능개선 원리

오라클 - 배치 I/O
- 테이블 블록에 대한 디스크 I/O call을 미뤘다가 읽을 블록이 일정량 쌓이면 한꺼번에 처리
- 블록마다 건건이  I/O call을 발생시키는 비효율을 줄이기 위한 기능
- 데이터 출력 순서가 인덱스 정렬 순서와 다를 수 있음
